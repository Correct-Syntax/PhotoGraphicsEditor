About

PhotoGraphics Editor is a free and open source image editing program written in the Python Programming Language.

This project's aim is to create a user-friendly, yet powerful and feature-rich image editor which allows Python programmers to easily write plugins for specific image manipulation functions, etc. built on this pre-built program framework.

PhotoGraphics Editor is currently available for Windows XP 32-bit OS and higher. The entire source code is available for download for free.

Version 1.0.0

PhotoGraphics Editor 1.0.0 is the first release after over a year of development. Version 1.0.0 adds the new Create Gradient option, an update of the previous Resize tool, major fixes to the Add Text tool UI, better stability and error handling, plus many bug fixes.

Features

PhotoGraphics Editor 1.0.0 provides many normal image-editing tools and a few new tools and implemetations of tools.

8 main tools have been implemented as of version 1.0.0, including Crop, Clone, Add Text, Layer, Generate Normal Map and more. PhotoGraphics Editor has 13 built-in filters and multiple options for opening, saving and creating images. Standard editing functions such as image transforms and converting between image modes are included with the program's integrated undo/redo cache system and interactive "flat-layout" tool option panel and UI. Create your own tools, filters or options by scripting a plugin for Photographics Editor using Python with the program's core and included libraries.

Bugs

This is considered to be a stable release, but you may encounter bugs. If you do, please help me by reporting the bug on the "Software Bugs" webpage. Please post detailed instructions on what image format was used and how to reproduce the error.

Installation

Windows: The downloaded .zip contains a "PhotoGraphics Editor" folder. You may put this anywhere on your hard drive*. To launch PhotoGraphics Editor, double-click on "PhotoGraphics Editor.exe".

* If you plan to create or install plugins, you may not be able to put the program on drive "C:\".

Installing Plugins: See the register_plugins.py file ("__plugins__">>"register_plugins") to learn how to create and register plugins.


Getting Started

When opening PhotoGraphics Editor, you will see a white "canvas" area in the center, a Toolbar on the left, Properties sidebar on the right and a Status-bar at the bottom.

Open an image into the editor through the menu (File >> Open..) or by pressing Ctrl+O. Edit the image by either choosing one of the tools in the Toolbar or selecting an option in the menu. Save the edited image through the menu (File >> Save As..) or by pressing Shift+S.

With the image loaded into the editor, select a tool filter or option from the menus. Edit the image by changing the values in the Properties sidebar.
